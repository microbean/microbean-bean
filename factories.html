Factories

When turning something like this:

@ApplicationScoped
@C
public class A {

  private B b;

  public A() {
    super();
  }
  
  @Inject
  public void setB(B b) {
    this.b = b;
  }

}

…into a Factory:

  • The instance type is A
  • The bean types are those of the transitive type closure of the instance type
  • The instance qualifiers are C
  • The factory class is A
  • The factory qualifiers are C
  • The factory executable is A()
  • The factory executable variables are none.
  • The initializer variables are the B-typed parameter of the setB(B) initializer method
  • The disposer executable is null
  • The disposer executable variables are none.
